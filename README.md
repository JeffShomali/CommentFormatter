###Assembly Comment FormmaterThis small bash program help you to format the comments in an assembly language source file.### Conditions:Comments begin with a ; (semicolon) and end with a newline.The source file will contain only printing characters, spaces, and newlines. It is not necessary to check for invalid input.### Usage./formatter.sh filename### Output![alt tag](https://raw.githubusercontent.com/JeffShomali/CommentFormatter/master/example.png)
